# CSS

## Center content in a div with flex

```text
div {
    display: flex;
    align-items: center;
    justify-content: center;
}

```

## Center content in absolute position

```text
div {
    position: absolute;
    margin-left: auto;
    margin-right: auto;
    left: 0;
    right: 0;
    text-align: center;
}
```
